# Android ADB Monitor

Простой инструмент для проверки основных параметров телефона через ADB.

## Возможности
Мониторинг в реальном времени с обновлением каждые 0.5 секунд.

Информация о батарее - уровень, тип, статус, температура, здоровье,

Информация о дисплее - разрешение и частота обновления,

Загрузка процессора,

Использование памяти,

Стильный интерфейс с темной темой,

Полноэкранный режим по F11

## Установка и использование
Требования: Python 3.8+, ADB, Android устройство с включенной отладкой по USB

Скачать ADB:
Windows: Скачайте Platform Tools с официального сайта
Linux: sudo apt install adb
macOS: brew install android-platform-tools

Запуск приложения:
python android_monitor.py

Подключение устройства:
adb devices
adb kill-server && adb start-server (если устройство не видно)
Разрешить отладку по USB на устройстве при запросе

Управление:
F11 - Переключение полноэкранного режима
Escape - Выход из полноэкранного режима
Крестик окна - Закрыть приложение

## Отслеживаемые параметры
Батарея: Уровень, технология, статус зарядки, температура, здоровье
Дисплей: Разрешение экрана и частота обновления
Процессор: Загрузка CPU
Память: Использование оперативной памяти и SWAP

## Решение проблем
Устройство не определяется:
Проверьте кабель USB
Подтвердите отладку по USB на устройстве
Выполните: adb kill-server && adb start-server

Ошибки при запуске:
Убедитесь что установлен Python 3.8+
Проверьте что ADB доступен в PATH
Убедитесь что включена отладка по USB в настройках разработчика
