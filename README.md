<p><strong><span style="font-size: 2em;">Android Diagnostic Board (ADB Monitor)</span></strong></p>

**Графическое приложение для мониторинга и диагностики Android-устройств через ADB.**

---
</div>

## 📋 Описание
Утилита отображает подробную системную информацию, показатели производительности в реальном времени и позволяет выполнять базовые операции управления подключенным гаджетом.

## ✨ Основные функции
* 📊 **Мониторинг системы:** CPU, RAM, встроенная память, статус батареи и параметры дисплея.
* 🔧 **Управление устройством:** быстрая перезагрузка (в том числе в Fastboot/Recovery) и выключение.
* ⚡ **Бенчмарк производительности:** встроенные тесты процессора, оперативной памяти и скорости ввода-вывода (I/O).
* 🔓 **Проверка загрузчика:** специализированный инструмент проверки статуса Bootloader (актуально для устройств Xiaomi).
* 📱 **Автоопределение:** автоматический вывод версии Android, точной модели устройства и его ключевых характеристик.

## ⚙️ Технические особенности
* **Кроссплатформенность:** полноценная работа на Windows, Linux и macOS.
* **Умный поиск:** автоматическое обнаружение установленного в системе ADB.
* **Удобный интерфейс:** поддержка полноэкранного режима по нажатию клавиши `F11`.
* **Динамика:** обновление всех параметров в режиме реального времени.
* **Стабильность:** подробное логирование внутренних ошибок для быстрой отладки.

## 📌 Требования
1. **Python 3.6** или выше.
2. Утилита **ADB** (должна быть добавлена в переменные среды PATH, если приложение не найдет её автоматически).
3. Подключенное Android-устройство с включенной **отладкой по USB** (USB Debugging).

## 🚀 Установка и запуск

Перед первым запуском убедитесь, что все необходимые библиотеки установлены. Откройте терминал/CMD в папке с проектом и выполните:
```bash
pip install -r requirements.txt
```

### Способы запуска:
* **Вариант 1:** Двойной клик по файлу `ADB_Monitor.py` (если в системе настроена ассоциация файлов `.py` с Python).
* **Вариант 2:** Откройте командную строку (CMD / Терминал) в папке приложения и введите команду:
  ```bash
  python ADB_Monitor.py
  ```
